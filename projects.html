Dog Classifier
Uses convolutional neural networks to create a program that detects the breed of a dog based on its physical appearance. If a human is detected, then the program will match the human face to the most similar dog breed. If no dog or human is detected then the program will not try to match the image to a dog breed. Any image input is accepted.
